# Real-Time Collabortive-Drawing Canvas

## Overview
A real-time multi-user drawing application where multiple users can draw simultaneously on a shared canvas and see each others drawing.

## Tech Stack
- React (Frontend)
- HTML5 Canvas API
- Node.js + Express
- Socket.io (WebSockets)

## Features
- Real-time drawing sync
- Multi-user support
- User-specific undo
- Raw Canvas API (no drawing libraries)

## Setup Instructions

### Install dependencies

  npm install
  cd client-react && npm install
